= Немного запятых непомешает

Напишите программу, которая принимает строку ненулевой длинны,
состоящую из символов 0-9, без ведущих нолей.
Программа должн вывести строку с правильно отформатированным числом

== Пример:
Input:
30000000

Output:
30,000,000


Input:
3474902022344556634

Output:
3,474,902,022,344,556,634
